# FullBand — landing site

The marketing site for **FullBand**, a local app that turns any song into a full
multitrack mixer (separate stems, per‑instrument faders, transpose/tempo, A–B
loop, record‑along, Stage mode).

**Live:** https://fullband.netlify.app

## Editing

It's a single self‑contained `index.html` (inline CSS + JS, no build step).
Edit it and open it in a browser to preview.

## Deploys

Every push to `main` **auto‑deploys** to Netlify via GitHub Actions
(`.github/workflows/deploy.yml` → the `fullband` Netlify site). Nothing to run
by hand.
